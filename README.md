# mehedi-cosmos

Immersive 3D portfolio — a cinematic interactive universe. Not a website.

- **Architecture:** see `docs/COSMOS_OS_ARCHITECTURE.md` (Phase 1 blueprint — single source of engineering truth)
- **Stack:** Next.js 15 · React 19 · Three.js · React Three Fiber 9 · GSAP · Zustand · Zod · GLSL
- **Principles:** Exploration over navigation · Story over resume · Physics over decoration · Nothing teleports

## Dev
```bash
npm install
npm run dev          # universe at localhost:3000
npm run typecheck
npm run content:validate   # Zod-validate world content (runs in CI)
```
